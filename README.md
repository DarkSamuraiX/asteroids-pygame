# Asteroids Game

A classic Asteroids arcade game implementation using Pygame.

## About

This is an Asteroids game project created as part of a Boot.dev learning task. The game recreates the iconic 1979 arcade experience where players control a spaceship to destroy asteroids and survive incoming threats.

## Features

- Classic asteroid-breaking gameplay
- Player ship controls and shooting mechanics
- Progressive difficulty as asteroids break apart
- Score tracking
- Collision detection

## Requirements

- Python 3.x
- Pygame

## Installation

```bash
pip install pygame
```

## Running the Game

```bash
python main.py
```

## Controls

- **WASD** - Rotate and move your ship
- **Space** - Shoot
- **ESC** - Quit game

## How to Play

1. Destroy all asteroids without colliding with them
2. Each destroyed asteroid may split into smaller pieces
3. Survive as long as possible and maximize your score
